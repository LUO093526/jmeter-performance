# 商城 JMeter 性能测试

课程实训项目，使用 JMeter 对商城系统进行接口测试与性能压测。

## 文件说明

| 文件 | 说明 |
|------|------|
| 性能测试.jmx | 1000并发性能压测，目标接口 GET /api/goods/getHostList，含聚合报告 |
| 购物流程.jmx | 完整购物流程接口自动化：登录→搜索→商品详情→加购→下单→支付 |
| 数据库压测.jmx | JMeter JDBC 直连 MySQL 数据库压测 |
| 查看结果11树.jmx | 数据库压测（含查看结果树监听器） |

## 技术点

- 线程组配置（1000线程，1秒爬坡）
- SyncTimer 同步定时器（分批释放请求）
- ConstantThroughputTimer 常量吞吐量定时器
- JSON提取器 / 正则提取器（接口关联）
- If 逻辑控制器（条件分支）
- JDBC Connection 连接池
- 聚合报告分析

## 使用方式

1. 安装 JMeter 5.6.3+
2. 安装 MySQL JDBC 驱动（数据库压测需要）
3. 修改目标服务器地址和端口
4. 修改数据库连接信息
5. 在 JMeter GUI 中打开 .jmx 文件运行
